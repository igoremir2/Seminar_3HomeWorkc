# Git. Руководство пользователя
## Основные команды
* git inint - инициализирует локальный репозиторий
* git commit - создает коммит
* git diff - разница между содержимым
* git merge - слияние
* git branch -d "ветка" - удаление ветки
* git branch "ветка" - создание ветки
